## Technology Stack

This platform is built using modern web technologies:

- **Framework**: Astro 5.x with static site generation
- **Styling**: Tailwind CSS for responsive design
- **Content**: MDX for blog posts and promotional content
- **Icons**: Astro Icon with Iconify integration
- **Integrations**: Sitemap, RSS feeds, and Google Reviews
